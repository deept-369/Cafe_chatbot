# ☕ Brewed Awakening Café – Chatbot Website

A full-stack café chatbot website built with **Python Flask**, **HTML/CSS**, and **JavaScript**.

## 📁 Project Structure

```
cafe_chatbot/
├── app.py                  # Flask backend + chatbot logic (30+ Q&A pairs)
├── requirements.txt
├── templates/
│   └── index.html          # Full café website with chatbot
└── static/
    ├── css/style.css       # Warm espresso-themed design
    └── js/chat.js          # Chat UI & API fetch logic
```

## 🚀 How to Run

```bash
# 1. Install dependencies
pip install -r requirements.txt

# 2. Start the server
python app.py

# 3. Open in browser
http://localhost:5000
```
